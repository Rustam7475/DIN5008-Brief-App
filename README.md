## 🇩🇪 Deutsch

### Beschreibung

**DIN 5008 Brief Editor** ist eine plattformübergreifende Desktop-Anwendung zur Erstellung professioneller Geschäftsbriefe nach der deutschen Norm **DIN 5008**. Die App unterstützt Form A, Form B und das C4-Format und bietet 13 fertige Briefvorlagen, Kontaktverwaltung, Mehrfachprofile, PDF-Export und vieles mehr.

### Funktionen

**Briefeditor**
- Geschäftsbrief-Layout nach DIN 5008 Form A, Form B und C4
- Millimetergenaue Positionierung aller Elemente auf A4
- Absender-Rücksendeangabe, Empfängerfeld, Informationsblock
- Betreff, Anrede, Fließtext, Grußformel, Unterschrift
- Fußzeile mit drei Spalten (Kontakt, Bank, Sonstiges)
- Falzmarken für Fensterbriefumschläge
- Rich-Text-Formatierung (fett, kursiv, unterstrichen, Listen)
- Zoom-Steuerung (Vergrößern / Verkleinern / An Seite anpassen)
- Live-Vorschau im Split-Pane-Editor

**13 Briefvorlagen**
- Kündigung, Mahnung, Widerspruch, Bewerbung, Reklamation
- Widerruf, Mietminderung, Abmahnung, DSGVO-Auskunft
- Vollmacht, Schadensersatz, Ratenzahlung, Mietkaution
- Eigene Vorlagen erstellen und speichern

**Profile & Kontakte**
- Mehrfachprofile (Privat, Geschäftlich usw.)
- Firmenlogo-Upload (PNG/JPG/SVG)
- Handunterschrift per Zeichenfeld (Canvas mit adaptiver Strichbreite)
- Kontaktverwaltung mit CSV- und JSON-Import/Export
- CC-Empfänger-Unterstützung

**Briefverwaltung**
- Gespeicherte Briefe mit Statusverwaltung (Entwurf / Gesendet / Archiviert)
- Suche und Filterung nach Status
- Briefe duplizieren
- Vollständiger Import/Export aller Daten (JSON)

**Export & Versand**
- PDF-Export (über jsPDF + html2canvas)
- E-Mail-Versand via mailto-Link
- Druckfunktion mit exakter DIN-5008-Positionierung

**Benutzeroberfläche**
- Dreisprachig: Deutsch, Russisch, Englisch
- Dunkelmodus
- Tastenkombinationen (⌘S Speichern, ⌘P Drucken, ⌘N Neu usw.)
- Onboarding-Assistent beim ersten Start
- Tooltips mit Erklärungen zu allen Feldern
- Barrierefreie Tastaturbedienung (Focus-Visible)
- Automatische Speicherung aller Eingaben (localStorage)

### Installation

#### macOS
1. Öffnen Sie **DIN 5008 Brief Editor-3.0.0-universal.dmg**
2. Ziehen Sie die App in den Ordner **Programme**
3. Beim ersten Start: Rechtsklick → **Öffnen** → bestätigen

#### Windows
1. Führen Sie **DIN 5008 Brief Editor Setup 3.0.0.exe** aus
2. Folgen Sie dem Installationsassistenten

#### Linux
1. Laden Sie **DIN 5008 Brief Editor-3.0.0.AppImage** herunter
2. `chmod +x` und ausführen

## 🇷🇺 Русский

### Описание

**DIN 5008 Brief Editor** — кроссплатформенное настольное приложение для создания профессиональных деловых писем по немецкому стандарту **DIN 5008**. Поддерживает форматы Form A, Form B и C4, включает 13 готовых шаблонов писем, управление контактами, мультипрофили, экспорт в PDF и многое другое.

### Возможности

**Редактор писем**
- Макет делового письма по DIN 5008: Form A, Form B и C4
- Точное позиционирование всех элементов на листе A4 (в миллиметрах)
- Обратный адрес отправителя, поле получателя, информационный блок
- Тема, обращение, основной текст, заключительная фраза, подпись
- Нижний колонтитул с тремя колонками (контакты, банк, прочее)
- Линии сгиба для оконных конвертов
- Форматирование текста (жирный, курсив, подчёркивание, списки)
- Управление масштабом (увеличение / уменьшение / по размеру страницы)
- Предпросмотр в реальном времени (разделённый редактор)

**13 шаблонов писем**
- Расторжение договора, напоминание об оплате, возражение, резюме, рекламация
- Отзыв, снижение арендной платы, предупреждение, запрос по DSGVO
- Доверенность, возмещение ущерба, рассрочка платежа, залог аренды
- Создание и сохранение собственных шаблонов

**Профили и контакты**
- Несколько профилей (личный, рабочий и т.д.)
- Загрузка логотипа компании (PNG/JPG/SVG)
- Рукописная подпись через поле рисования (Canvas с адаптивной толщиной)
- Управление контактами с импортом/экспортом CSV и JSON
- Поддержка CC-получателей

**Управление письмами**
- Сохранённые письма со статусами (Черновик / Отправлено / Архив)
- Поиск и фильтрация по статусу
- Дублирование писем
- Полный импорт/экспорт всех данных (JSON)

**Экспорт и отправка**
- Экспорт в PDF (через jsPDF + html2canvas)
- Отправка по email через mailto-ссылку
- Печать с точным соблюдением позиций DIN 5008

**Интерфейс**
- Три языка: немецкий, русский, английский
- Тёмная тема
- Горячие клавиши (⌘S сохранить, ⌘P печать, ⌘N новое письмо и др.)
- Мастер первого запуска (Onboarding)
- Подсказки с пояснениями к каждому полю
- Доступность с клавиатуры (Focus-Visible)
- Автоматическое сохранение всех данных (localStorage)

## 🇬🇧 English

### Description

**DIN 5008 Brief Editor** is a cross-platform desktop application for creating professional business letters according to the German standard **DIN 5008**. It supports Form A, Form B and C4 formats, includes 13 ready-made letter templates, contact management, multiple profiles, PDF export and more.

### Features

**Letter Editor**
- Business letter layout per DIN 5008: Form A, Form B and C4
- Millimeter-precise positioning of all elements on A4
- Sender return address, recipient field, information block
- Subject, salutation, body text, closing phrase, signature
- Footer with three columns (contact, bank, other)
- Fold marks for window envelopes
- Rich text formatting (bold, italic, underline, lists)
- Zoom controls (zoom in / zoom out / fit to page)
- Live preview in split-pane editor

**13 Letter Templates**
- Cancellation, payment reminder, objection, job application, complaint
- Withdrawal, rent reduction, warning, GDPR data request
- Power of attorney, damages claim, installment payment, rental deposit
- Create and save custom templates

**Profiles & Contacts**
- Multiple profiles (personal, business, etc.)
- Company logo upload (PNG/JPG/SVG)
- Handwritten signature via drawing canvas (adaptive stroke width)
- Contact management with CSV and JSON import/export
- CC recipient support

**Letter Management**
- Saved letters with status tracking (Draft / Sent / Archived)
- Search and filter by status
- Duplicate letters
- Full import/export of all data (JSON)

**Export & Sending**
- PDF export (via jsPDF + html2canvas)
- Email sending via mailto link
- Print with exact DIN 5008 positioning

**User Interface**
- Trilingual: German, Russian, English
- Dark mode
- Keyboard shortcuts (⌘S save, ⌘P print, ⌘N new letter, etc.)
- Onboarding wizard on first launch
- Tooltips with explanations for all fields
- Keyboard accessibility (focus-visible)
- Auto-save all inputs (localStorage)

### Installation
